# PVZ AI 游戏数据仓库

这是一个包含植物大战僵尸AI版完整游戏配置数据的TypeScript文件仓库。

## 📁 文件结构

```
pvz-ai-game-data/
├── src/data/
│   └── gameData.ts    # 完整的游戏配置数据
└── README.md          # 项目说明文档
```

## 🎮 游戏内容概览

### 植物类型 (39种)
- **人类世界植物**: 清鸢、善•猫教主、徐清岚等
- **光明大陆植物**: 各种光明系法师和治疗系植物
- **暗影大陆植物**: 暗影系刺客和召唤师
- **洪荒古神类**: 离火双生•长离、镜花水月·灵汐、苍玄等强大古神植物

### 僵尸类型 (24种)
- **普通僵尸**: 各类基础僵尸兵种
- **精英僵尸**: 具有特殊能力的强化僵尸
- **Boss级僵尸**: 寂灭之主•玄烬、冰凝神祇等终极Boss

### 关卡配置
- **主线关卡**: 1-10关完整配置
- **无尽模式**: 高难度挑战设置
- **特殊规则**: 各种游戏机制和限制

### 成就系统
- **核心成就**: 基础游戏进度成就
- **成长成就**: 角色培养相关成就
- **战斗成就**: 战斗技巧相关成就
- **收集成就**: 收集元素相关成就
- **隐藏成就**: 特殊触发条件成就

## 🚀 使用说明

1. 直接引用 `src/data/gameData.ts` 文件作为游戏配置
2. 在TypeScript项目中导入所需的游戏数据
3. 根据游戏需求选择对应的植物和僵尸配置
4. 参考成就系统和关卡配置设计游戏流程
